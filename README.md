# Expense Analysis App

A simple **Streamlit-based Expense Analysis application** that helps visualize and analyze personal expenses over time using Python and Pandas.

---

## Features

- Upload and analyze expense data
- Monthly expense aggregation
- Interactive line chart visualization using Streamlit
- Clean and simple UI
- Jupyter Notebook included for data exploration

---

## Tech Stack

- **Python 3**
- **Streamlit**
- **Pandas**
- **Matplotlib**
- **Jupyter Notebook**
